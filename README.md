## NSE cli in go language

This repository contains nse api to run as cli fully written in go language

## Clone the project

```
$ git clone https://github.com/MdTosif/go-nse
$ cd go-nse
```

creating a build to run

```
$ cd go-nse
$ go build
$ ./nse help
```
